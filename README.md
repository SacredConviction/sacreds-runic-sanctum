This module contains three compendiums, Sacred's Runic Sanctum(How Runes Work) contains a journal that explains the functions of Runes and how they can be applied, removed, and stacked. Sacred's Runic Sanctum(Roll Tables), currently only contains one roll table that is used for one rune which is the Wild Magic Surge table. Sacred's Runic Sanctum(Rune Chisels), contains the heart of the module, the Rune Chisels. Items that can be given to players and then applied to equipment through enchanting
